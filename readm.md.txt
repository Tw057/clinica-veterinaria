# 🐾 Clínica Veterinária - Sistema de Gestão

Sistema web completo para gerenciamento de atendimentos em uma clínica veterinária.

![Status](https://img.shields.io/badge/status-online-brightgreen)

---

## 🚀 Demonstração

🔗 Acesse o sistema online:  
(https://clinica-veterinaria-kp0e.onrender.com/)

---

## 📸 Funcionalidades

- ✅ Cadastro de clientes
- ✅ Cadastro de pets
- ✅ Agendamento de atendimentos
- ✅ Histórico de atendimentos por pet
- ✅ Dashboard com últimos atendimentos
- ✅ Controle de serviços ativos

---

## 🛠 Tecnologias Utilizadas

- PHP (POO)
- PostgreSQL
- Bootstrap 5
- Render (Deploy)
- Git e GitHub

---

## 🧠 Aprendizados

Este projeto foi desenvolvido com foco em:

- Organização em camadas (classes, controladores)
- Integração com banco de dados real (PostgreSQL)
- Tratamento de erros com PDO
- Deploy de aplicação web

## ⚙️ Como rodar localmente

1. Clonar o repositório:

git clone https://github.com/seuusuario/clinica-veterinaria.git


2. Configurar o banco de dados no arquivo:

config/database.php


3. Rodar o projeto em servidor local (XAMPP/Laragon)
