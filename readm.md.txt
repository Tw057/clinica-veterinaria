# 🐾 Sistema de Clínica Veterinária

Sistema web completo para gerenciamento de clínica veterinária.

## 🚀 Funcionalidades

* Cadastro de clientes
* Cadastro de pets
* Agendamento de atendimentos
* Histórico de atendimentos

## 🛠️ Tecnologias

* PHP
* PostgreSQL
* Docker
* Bootstrap
* Render (Deploy)

## 🌐 Acesso ao sistema

👉 https://SEU-LINK.onrender.com

## 📂 Código-fonte

👉 https://github.com/Tw057/clinica-veterinaria

## 💻 Como rodar localmente

```bash
git clone https://github.com/Tw057/clinica-veterinaria.git
cd clinica-veterinaria
php -S localhost:8000
```
